## Why Signal matters

**[[Gold]] standard for encryption:** Signal Protocol used by WhatsApp, Facebook Messenger, [[Google]] Messages, Skype.

**No business model:** No ads, no data collection, no monetization. Runs on donations.

**Journalist/activist tool:** Default for sensitive communications.

---

## Features

- **End-to-end encryption** — all messages, calls
- **Disappearing messages** — auto-delete
- **Sealed sender** — metadata protection
- **No phone number** — username support added
- **Open source** — auditable code
