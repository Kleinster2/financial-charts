**Ampere Computing** — ARM-based server chip designer. **Acquired by [[SoftBank]] for $6.5B (2025).**

---

## SoftBank acquisition (2025)

| Detail | Value |
|--------|-------|
| Acquirer | [[SoftBank]] |
| Price | **$6.5B** |
| Strategic fit | Arm + Ampere = vertical integration |

**Rationale:** SoftBank owns ~90% of [[Arm Holdings]]. Ampere designs ARM-based server chips. Acquisition creates vertical stack for AI inference and data center computing.

| Metric | Value |

## Market position

**Competing with:**
- [[Intel]] Xeon (x86)
- [[AMD]] EPYC (x86)
- AWS Graviton (in-house ARM, not sold)
- [[NVIDIA]] Grace (ARM, AI-focused)

**Advantage:** Pure-play ARM server chips for cloud workloads. Power efficiency matters for AI inference at scale.
