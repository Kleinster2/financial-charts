# Runway

AI video generation company. Founded 2018. Pioneered commercial text-to-video. Gen-3 Alpha (2024) and Gen-4 (2025) set creative video standards. Hollywood partnerships.

| Founded | 2018 |
| HQ | New York |
| Funding | ~$555M+ |
| Valuation | **$5.3B** (Series E, Feb 2026) |
| Focus | Video generation, creative tools |
