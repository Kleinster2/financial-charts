## Product

### How it works

1. Describe app in natural language
2. Bolt.new generates full-stack code using [[Claude]] 3.5 Sonnet
3. Runs in browser via [[StackBlitz]] WebContainer
4. One-click deploy

### Tech stack

| Component | Technology |
|-----------|------------|
| AI model | [[Claude]] 3.5 Sonnet ([[Anthropic]]) |
| Runtime | StackBlitz WebContainer |
| Frontend | React, Vue, Svelte supported |
| Backend | Node.js in browser |

### Pricing

| Plan | Price | Tokens |
|------|-------|--------|
| Free | $0 | 1M tokens |
| Pro | $20/mo | 10M tokens |
| Teams | Custom | — |

**Token model:** Pay for AI usage, not seats. Aligns cost with value.
