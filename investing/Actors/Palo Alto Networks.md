## SASE stack

Palo Alto's [[SASE]] posture is the cleanest example of bundled SASE in the vault. Prisma Access provides cloud-delivered secure access; Prisma SD-WAN pulls branch connectivity into the same sale; Prisma Access Browser, Prisma Agent, SWG, CASB, FWaaS, and ZTNA extend policy into web, SaaS, private-app, and browser workflows. The strategy is not to win secure access as a point product. It is to make SASE another module in the broader platformization account motion.

| Layer | Palo Alto product surface | Function |
|-------|---------------------------|----------|
| Private-app access | Prisma Access / ZTNA | Least-privilege access without broad VPN-style network reach |
| Web and SaaS | SWG, CASB, DLP, RBI | Controls browsing, cloud-app usage, and sensitive data movement |
| Branch / WAN | Prisma SD-WAN and remote networks | Connects branches and sites through the SASE fabric |
| Firewall policy | FWaaS / NGFW heritage | Moves Palo Alto inspection depth into cloud-delivered enforcement |
| Telemetry / experience | ADEM, Prisma Access Browser, Prisma Agent | Makes secure access observable and easier to operate |

*Sources: [Palo Alto Networks Prisma Access](https://www.paloaltonetworks.com/sase/access); [Palo Alto Networks SASE architecture guide](https://www.paloaltonetworks.com/cyberpedia/sase-architecture).*

## May 2026 — Zscaler SASE read-through

Reuters' May 26 coverage of [[Zscaler]]'s Q3 FY2026 guide named Palo Alto's platform offerings as a competitive pressure in [[SASE]]. The read-through is not a Palo Alto earnings event; it is validation that Prisma SASE and broader platformization are becoming share-of-wallet threats even for the zero-trust pure-play. If customers consolidate secure access, cloud security, endpoint telemetry, and SOC automation with fewer vendors, Palo Alto's pitch improves; if best-of-breed architecture wins, Zscaler keeps more of the SASE budget.

*Source: [Reuters via StreetInsider, May 26 2026](https://www.streetinsider.com/Reuters/Zscaler%2Bsees%2Bdownbeat%2Bquarterly%2Brevenue%2Bas%2Bcompetition%2Bheats%2Bup%2Bin%2Bcybersecurity%2Bmarket/26550638.html).*

## China cybersecurity ban (Jan 2026)

Banned by Beijing — China instructed companies to stop using Palo Alto Networks products and replace with domestic alternatives by H1 2026.

Rationale: Beijing fears Western cybersecurity software could collect and transmit confidential information abroad.

Other banned US firms: VMware, Fortinet, Mandiant, CrowdStrike, [[SentinelOne]], McAfee, Recorded Future, Claroty, Rapid7, Wiz

## CyberArk acquisition ($25B, closed Feb 11 2026)

Largest transaction in cybersecurity history. [[CyberArk]] shareholders received $45.00 cash + 2.2005 PANW shares per CYBR share. Shareholders approved Nov 13, 2025.

Strategic rationale: machine identities now outnumber human identities 80:1. 75% of organizations have overly permissive privilege models. ~90% have suffered identity-centric breach. CyberArk's PAM, identity provider, and governance tools integrate into Prisma and Cortex platforms.

Regulatory: behavioral remedies required — CyberArk tools must remain interoperable with third-party security vendors (no walled garden).

Planning TASE secondary listing under ticker "CYBR" as tribute to CyberArk's Israeli heritage.

Acquired [[Chronosphere]], a cloud-native observability platform, for over $3B. Extends monitoring and observability capabilities across the security platform — part of the broader strategy to own the full security operations stack.

## Koi acquisition ($400M, Feb 2026)

Acquiring Israeli startup Koi for ~$400M to secure "agentic endpoints" — AI agents and tools that operate with deep data access but bypass traditional security controls.

| Detail | Value |
|--------|-------|
| Price | ~$400M (reported) |
| Founded | 2024 |
| Founders | [[Unit 8200]] alumni (Amit Assaraf CEO) |
| Problem | AI agents create unmanaged attack surface on endpoints |
| Integration | Extends Prisma AIRS + Cortex XDR |

The thesis: as enterprises deploy AI agents ([[Agentic AI]]), traditional file-based security misses agent-level threats — credential hijacking, API-based RCE, spoofed agent identities. Koi replaces download buttons with approval workflows and evaluates file/developer risk.

This is Palo Alto's latest in a string of acquisitions building out its AI security platform, following the [[Cybersecurity consolidation]] pattern.
