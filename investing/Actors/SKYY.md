# SKYY

SKYY (Nasdaq: SKYY) is the First Trust Cloud Computing ETF — the oldest and most established US-listed [[Cloud computing|cloud-computing]] theme fund, holding infrastructure providers, platform/software names, and the hyperscaler parents across the cloud value chain. Launched 2011, it is the canonical investable proxy for the cloud theme (the WisdomTree WCLD is a software-tilted alternative).
