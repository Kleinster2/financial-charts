# Kohl's

Kohl's (NYSE: KSS) is a mid-tier US department store in a multi-year turnaround — declining traffic, margin pressure, and recurring strategic/leadership churn, with the equity (~$17) trading as a deep-value/distressed retail name. Used as a full-price department-store control in the [[Off-price retail]] cluster test (against which off-price is supposed to be counter-cyclically distinct).

## Use case

A department-store control for the [[Off-price retail#Cluster validation|off-price cluster validation]].
