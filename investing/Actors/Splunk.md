# Splunk

Security information and event management ([[SIEM]]) + observability platform. Acquired by [[Cisco]] for $28B (announced Sep 2023, closed Mar 2024) — Cisco's largest-ever acquisition. Operates as wholly-owned subsidiary inside Cisco's Security & Collaboration segment.
