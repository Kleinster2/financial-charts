## Product

Adallom secured enterprise SaaS applications:
- Audited user activity
- Protected against cloud threats
- Visibility into shadow IT

Early cloud access security broker (CASB).

---

## Microsoft integration

| Date | Event |
|------|-------|
| July 2015 | Acquired for ~$320M |
| April 2016 | Launched as Microsoft Cloud App Security (MCAS) |
| Nov 2021 | Rebranded to Microsoft Defender for Cloud Apps |

Founders removed ~75% of features to make it scalable, deployed to 100M+ users.

---

## Post-acquisition

Rappaport became head of Microsoft Israel R&D (~1,500 employees) and led Cloud Security Group.

After 4 years at Microsoft, the four founders left to start [[Wiz]] (January 2020).
