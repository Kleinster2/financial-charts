# Boris Cherny

[[Anthropic]] engineer, creator and head of [[Claude Code]]. In 2024, sent a Slack message to colleagues: "I wanted to show off a new tool I've been hacking on." The tool became Claude Code — Anthropic's breakout product, hitting $2.5B run rate by Feb 2026 and driving more than half of Anthropic's enterprise revenue.

## Origin of Claude Code

Cherny built Claude Code as a side project and shared it on Slack in 2024. The tool spread organically inside [[Anthropic]] — "before long, everyone inside the company was using his new tool" (WSJ). Anthropic's existing focus on coders and enterprise customers meant Cherny had an unusually clear sense of who the users were when he built it. The product launched publicly in May 2025 and became Anthropic's fastest-growing product.

WSJ's Ben Cohen (Mar 20, 2026): Claude Code "conquered Silicon Valley, jolted Wall Street and even managed to scramble Anthropic's fiercest rival."

*Source: WSJ (Ben Cohen, Mar 20 2026)*
