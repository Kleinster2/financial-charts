## Memory cost impact

**CEO Enrique Lores (Nov 2025):**
- H2 2026 "particularly challenging"
- Memory accounts for **15-18%** of typical PC cost
- Will increase prices where necessary

**Mitigation actions:**
- Bringing on more memory suppliers
- Putting less memory in products (spec reduction)
- "Aggressive actions" to manage costs

## Investment case

**Bear:**
- Memory costs compressing margins
- PC demand soft
- Cost pass-through risks demand
- Less scale than Lenovo

**Bull:**
- Printer business less memory-exposed
- Can pass costs to enterprise
- Scale advantages vs smaller players
- Memory normalizes eventually
