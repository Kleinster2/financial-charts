# SAP

**World's largest ERP software company** — founded 1972 by five ex-IBM engineers in Germany. **NYSE: SAP**, XETRA: SAP. ~€340B market cap (~$325B) — largest European company by market cap. €35B revenue. Cloud transition via S/4HANA driving re-rating; 2027 ECC end-of-support forcing migrations.

## History

| Year | Event |
|------|-------|
| 1972 | Founded by 5 ex-IBM engineers in Weinheim |
| 1973 | First product: MIAS (materials, info, accounting) |
| 1980 | Moved to Walldorf; 80 employees |
| 1992 | R/3 launch — client/server ERP |
| 1998 | Hopp steps down as CEO |
| 2010 | Acquired Sybase ($5.8B) |
| 2012 | Acquired Ariba ($4.3B), SuccessFactors ($3.4B) |
| 2015 | S/4HANA launch — next-gen in-memory ERP |
| 2018 | Acquired Qualtrics ($8B) |
| 2024 | Hasso Plattner steps down from supervisory board |
| 2027 | **ECC end-of-support** — migration deadline |

## S/4HANA cloud transition

| Metric | Value |
|--------|-------|
| S/4HANA market (2024) | $20.4B |
| S/4HANA market (2033E) | $48.5B |
| CAGR | 10.1% |
| ECC customers | ~35,000 |
| Migrated to S/4HANA | ~14,000 (39%) |
| Holdouts by 2027 | ~17,000 (48%) |

**The forcing function:** SAP ending support for legacy ECC systems in 2027. Half of customers still haven't migrated — creates sustained demand tailwind through 2030.

## Investment relevance

**Why SAP matters:**

1. **ERP monopoly** — Largest ERP vendor globally; deep enterprise lock-in.

2. **Forced migration** — 2027 ECC end-of-support creating multi-year demand tailwind.

3. **Cloud transition** — 87% predictable revenue; margins expanding.

4. **Europe's tech champion** — Largest European company by market cap.

5. **AI optionality** — Joule and embedded AI across product suite.
