**OECD** — Organisation for Economic Co-operation and Development. 38 wealthy democracies. Sets global standards on tax, corporate governance, ESG, AI, education. Headquarters: Paris.

## Why OECD matters for investors

### 1. Global minimum tax (Pillar 2)

| Element | Detail |
|---------|--------|
| Rate | 15% minimum effective tax |
| Threshold | Companies with €750M+ revenue |
| Mechanism | Top-up tax if below 15% |
| Status | Rolling out 2024-2025 |
| Scope | 140+ countries agreed |

**Impact:** Erodes tax havens (Ireland, Netherlands, Luxembourg, Singapore, Cayman).

| Affected | Impact |
|----------|--------|
| Big Tech | Higher effective rates |
| Pharma | IP-based structures challenged |
| Multinationals | Restructuring needed |
| Ireland | 12.5% rate no longer competitive |

### 2. Pillar 1 (profit reallocation)

| Element | Detail |
|---------|--------|
| [[Target]] | Largest, most profitable MNCs |
| Threshold | €20B revenue, 10% margin |
| Reallocation | 25% of profits above 10% to market countries |
| Status | Stalled (US resistance) |

**Impact if enacted:** Big Tech, luxury, pharma pay more where customers are.

### 3. Transfer pricing guidelines

| Principle | Meaning |
|-----------|---------|
| Arm's length | Price as if unrelated parties |
| BEPS | Base Erosion and Profit Shifting rules |
| Documentation | Country-by-country reporting |
| Disputes | Mutual agreement procedures |

Sets global standard for how MNCs price intercompany transactions.

### 4. Anti-bribery convention

| Feature | Coverage |
|---------|----------|
| Scope | Bribery of foreign officials |
| Enforcement | National laws required |
| Monitoring | Peer reviews |
| Standard | Model for FCPA, [[UK]] Bribery Act |
