**GitHub** is the dominant code hosting, version control, and developer collaboration platform, with 100M+ developers and 420M+ repositories as of 2024. Acquired by [[Microsoft]] in 2018 for $7.5B — a deal widely criticized at the time but now considered one of the best enterprise acquisitions in tech history. GitHub generates an estimated $2B+ in annual revenue through GitHub Enterprise subscriptions, and its [[GitHub Copilot]] AI coding assistant (built on [[OpenAI]] models) has become the fastest-growing product in [[Microsoft]] history with 1.8M+ paid subscribers by early 2025.

## Synopsis

GitHub's investment relevance extends far beyond its direct revenue. It is the distribution layer for the entire open-source ecosystem and the default platform for AI model hosting, making it a critical piece of [[Microsoft]]'s AI strategy alongside [[Azure]] and the [[OpenAI]] partnership. [[GitHub Copilot]], priced at $10-39/month, represents the first mass-market AI developer tool and establishes recurring per-seat revenue from the global developer base. The competitive moat is network effects: developers go where the code is, and the code is overwhelmingly on GitHub.

CEO [[Thomas Dohmke]] (since 2021) has oriented the platform toward AI-native development, with Copilot integrated into the core workflow. Competitors include [[GitLab]] (DevOps-focused, public as GTLB), [[Atlassian]]'s [[Bitbucket]], and to a lesser extent self-hosted solutions — but none approach GitHub's scale or mindshare.
