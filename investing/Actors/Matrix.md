## What is Matrix

**Decentralized protocol** — like email for messaging. Anyone can run a server, servers federate.

| Component | Role |
|-----------|------|
| Protocol | Open standard (matrix.org) |
| Servers | Synapse (reference), Conduit, Dendrite |
| Clients | Element (flagship), FluffyChat, others |
| Foundation | Matrix.org Foundation (nonprofit) |

## Why Matrix matters

**Sovereignty:** Governments won't trust Slack/Teams for classified comms. Matrix = self-hosted, auditable.

**Interoperability:** EU Digital Markets Act may require WhatsApp/iMessage to interoperate. Matrix is leading candidate protocol.

**Open source:** No vendor lock-in.
