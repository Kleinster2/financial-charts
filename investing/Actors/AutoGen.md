## Architecture

Agents communicate via messages, each with defined roles:
- **AssistantAgent** — LLM-powered agent
- **UserProxyAgent** — Human-in-the-loop or automated executor
- **GroupChat** — Multi-agent conversation manager

Supports any LLM backend (OpenAI, Azure, local models).

---

## Why it matters

Microsoft's bet on multi-agent as the next paradigm. Competes with CrewAI, LangGraph. Enterprise integration potential via Azure.
