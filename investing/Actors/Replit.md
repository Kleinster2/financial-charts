## Product

### Replit Agent

AI coding agent that builds full applications from prompts. Key differentiator vs [[Cursor]]: runs entirely in browser, no local setup.

| Feature | Description |
|---------|-------------|
| Browser IDE | No installation, works on any device |
| AI Agent | Builds apps from natural language |
| Deployments | One-click hosting included |
| Multiplayer | Real-time collaboration |
| Ghostwriter | AI code completion |
