**CrewAI** — Open-source framework for orchestrating role-playing AI agents. Agents have defined roles, goals, and tools; collaborate to complete tasks.

## Architecture

| Concept | Description |
|---------|-------------|
| **Agent** | Autonomous unit with role, goal, backstory |
| **Task** | Specific assignment for an agent |
| **Crew** | Team of agents working together |
| **Process** | Sequential or hierarchical execution |

Designed for role-based collaboration — e.g., "researcher" agent works with "writer" agent.

---

## Why it matters

Simpler API than AutoGen. Popular for prototyping multi-agent systems. Competes with Microsoft AutoGen, LangGraph.
