## Synopsis

Faulkner is relevant to the vault not as a traditional executive but as a proof point for the AI-augmented developer thesis. His Vinext project — a ground-up reimplementation of Next.js's core functionality — would traditionally require a team of engineers working for months. He did it solo in a week, using [[Claude]] as a coding partner across hundreds of iterative sessions. The $1,100 token cost is the data point that matters: it puts a price tag on what was previously an abstract claim about AI developer productivity.

At [[Cloudflare]], Faulkner previously led the OpenNext integration effort, which aimed to make Next.js applications run natively on Cloudflare's edge network. The Vinext project emerged from that work — when integrating someone else's framework proved too constraining, he used AI to build an alternative from scratch. For investors in developer tooling and cloud infrastructure, the implication is structural: if a single engineer with AI tools can replicate the functionality of an open-source framework in days, the moat around framework-level software is thinner than previously assumed. The real defensibility moves to the platform layer (where [[Cloudflare]], [[Vercel]], and [[AWS]] compete) and to the data/network-effects layer.
