#actor #foundation #crypto #ai

**Opentensor Foundation** — Non-profit that develops [[Bittensor]], the decentralized AI protocol underlying the TAO token and its subnet ecosystem. Maintains the Subtensor blockchain substrate, [[Yuma Consensus]] reward algorithm, and core tooling. Operates the Proof-of-Authority validator set for the base chain.

| Metric | Value |
