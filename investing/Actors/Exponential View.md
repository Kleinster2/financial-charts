## Vault relevance

Exponential View is most useful when it separates technical capability from institutional adoption. The recurring edge is not "AI is powerful"; it is the question of which workflows, decision rights, incentives, and bottlenecks must change before model-level gains become firm-level productivity.

## Frameworks

- Installation is not transformation: deploying AI tools can raise local output while leaving the firm's decision bottlenecks unchanged.
- Stage 2 automation can look impressive in support, recruiting, sales, or coding while still preserving the old organizational geometry.
- Stage 3 value requires a new operating model where AI observes signals, orients against context, decides within governed boundaries, and shortens cycle time across workflows.
