#actor #crypto #ai #alliance

**ASI Alliance** (Artificial Superintelligence Alliance) — Multi-project merger that combined [[Fetch.ai]] (autonomous agents), SingularityNET (AI services marketplace), and Ocean Protocol (data monetization) into a single tokenized organization. Merger announced March 2024, token migration finalized July 15, 2024. Combined valuation at signing: $7.5B across 2.631B tokens. Current (April 2026) market cap: ~$518M — meaningful drawdown from peak. Covers agent + data layers of the [[Decentralized AI landscape]].

## Stack role

| Layer | Contribution |
|-------|--------------|
| Agent layer | [[Fetch.ai]] autonomous agent framework |
| AI services layer | SingularityNET decentralized AI services marketplace |
| Data layer | Ocean Protocol data monetization primitives |

The original thesis was that combining these three verticals would produce a coherent "decentralized AI stack" under one token. In practice, the merger has not yet produced unified products competitive with [[Bittensor]]'s subnet model or narrower purpose-built protocols like [[Morpheus]] and [[Virtuals Protocol]].

## Relevance

ASI is the "old guard" of decentralized AI — each constituent protocol predates the current cycle (Fetch since 2017, SingularityNET since 2017, Ocean since 2017). The merger represents an attempt to consolidate the pre-[[Bittensor]] generation into a unified vehicle. Market has given it a haircut from peak as newer, more focused protocols have taken mindshare.
