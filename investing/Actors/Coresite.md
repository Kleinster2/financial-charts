## Business model

| Service | Description |
|---------|-------------|
| Colocation | Rack space, power, cooling |
| Interconnection | Cross-connects, cloud on-ramps |
| Cloud on-ramps | Direct connects to AWS, Azure, GCP |
| Network density | High interconnection value |

**Differentiation:** Dense interconnection hubs — valuable for low-latency connections.

## Acquisition by American Tower

**2021 deal:**
- American Tower acquired Coresite for ~$10B
- Tower company → data center expansion
- Coresite operates as subsidiary
- No longer separately traded
