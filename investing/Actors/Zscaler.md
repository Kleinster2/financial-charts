## Why zero trust

**Legacy model broken:**
- Perimeter security assumes trusted network
- Remote work killed the perimeter
- Cloud apps bypass traditional firewalls

**Zero trust model:**
- Never trust, always verify
- User-to-application, not network-to-network
- Cloud-native, scalable
