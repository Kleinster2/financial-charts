## How to use in vault

TrendForce data useful for:
- Memory cycle timing (price inflection points)
- Validating [[SK Hynix]], [[Samsung]], [[Micron]] margin trends
- Display panel cost trends for TV/smartphone OEMs

**Caveat**: Some data behind paywall. Public reports often lagged.
