**GitHub Copilot** — AI pair-programmer from [[GitHub]] (owned by [[Microsoft]]), launched June 2021 as the first mainstream commercial coding assistant. Built initially on OpenAI Codex; now runs multiple frontier models under the hood (GPT-4-class, Claude, Gemini) with GitHub-side routing. Embedded in VS Code, Visual Studio, JetBrains, Neovim.

## Competitive position

Copilot was first to market and still has the largest installed base thanks to GitHub distribution, but has been increasingly pressured on capability by [[Cursor]] (purpose-built IDE with better agent UX) and on cost by Bittensor-native efforts like [[Ridges AI]]. [[Microsoft]] positions Copilot as the default assistant layer for Windows and the M365 suite, not just code.
