## Technology — WALL-A

X Square's proprietary embodied AI system:

| Component | Function |
|-----------|----------|
| **VLA model** | Vision-language-action |
| **World model** | Predicts actions, causal reasoning |
| **Integration** | VLA + world model architecture |

**Key capability:** Generalizes to new tasks without prior training. Improves mobile manipulation in complex, unstructured environments by using world models for action prediction and feedback interpretation.
