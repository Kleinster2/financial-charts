#actor #ai #search #agentic #private #usa

Firecrawl is a web-crawling and extraction API that turns websites into clean, LLM-ready data (markdown/structured output) for AI applications, with an open-source core and a popular developer following. It competes with [[Parallel Web Systems]], [[Exa]], and [[Tavily]] in the web-infrastructure-for-agents category, and scored in the statistically-tied top tier with Parallel on the independent AIMultiple search-quality benchmark. It was founded in 2022 by Caleb Peffer, Eric Ciarla, and Nicolas Silberstein Camara, emerging from the team's prior product Mendable (AI chat for docs, used by Snapchat, MongoDB, DoorDash).
