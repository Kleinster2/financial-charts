**London smart city** — \#6 global smart city index. Pioneer of congestion charging (2003), contactless transit (Oyster), and ULEZ. TfL planning £1.5B network overhaul. Waymo autonomous trials 2026.

## Key systems

### Congestion Charge

| Metric | Value |
|--------|-------|
| Launched | 2003 (first major city) |
| Current rate | **£18/day** (Jan 2026) |
| Previous rate | £15 (2020-2025) |
| Technology | ANPR cameras |
| Impact | Reduced central traffic ~15% |

**2026 changes:** EV discounts (50% vans, 25% cars) to manage rising EV uptake.

### Oyster & Contactless

| Metric | Value |
|--------|-------|
| Oyster launched | 2003 |
| Contactless | 2014 |
| Daily users | [[3M]]+ (subway alone) |
| Technology exports | NYC, Boston, Sydney |

### Ultra Low Emission Zone (ULEZ)

| Metric | Value |
|--------|-------|
| Launched | 2019 (central), 2023 (expanded) |
| Coverage | Greater London |
| Technology | ANPR network |
| Compliance | Non-compliant pay £12.50/day |
