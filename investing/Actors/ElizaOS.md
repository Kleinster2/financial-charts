#actor #crypto #ai #agents #framework

**ElizaOS** (formerly ai16z) — Open-source framework for building autonomous AI agents across blockchains. Launched as ai16z in 2024 — a DAO managed by an AI agent modeled on [[Andreessen Horowitz]]'s investment process. Peaked at $2.5B+ market cap. Rebranded to ElizaOS in early 2025 to broaden scope as a general-purpose agent framework; AI16Z token migrating 1:1 to ELIZAOS token. Primary competitor to [[Virtuals Protocol]] on the agent-platform layer.

## What it is

ElizaOS (software) is the open-source framework developers use to build custom agents. ai16z (DAO) is the flagship implementation — a token-governed DAO with an on-chain AI agent making investment decisions. Each builds the other's legitimacy: the framework proves itself through the DAO's live track record; the DAO proves itself by selecting good investments using the framework.

## Differentiation vs Virtuals

| Dimension | [[Virtuals Protocol]] | ElizaOS |
|-----------|----------------------|---------|
| Focus | Agent launchpad (issue tokens for agents) | Agent framework (build / deploy agents) |
| Chain | [[Base]] | [[Solana]], cross-chain |
| Economic model | Per-agent bonding curves | Single framework token + individual agents |
| Analogy | [[Shopify]] for agent tokens | Linux for agents |

Virtuals is primarily a financial primitive; ElizaOS is primarily a developer primitive. Both will likely co-exist serving different ecosystems.
