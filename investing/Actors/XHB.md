# XHB

XHB (NYSE Arca: XHB) is the State Street SPDR S&P Homebuilders ETF — an equal-weighted basket of the US homebuilding complex. Despite the name it is only ~30–35% homebuilders; the rest is building products, building materials, home-improvement retail, and home furnishings, so it functions as the broad residential-construction proxy.
