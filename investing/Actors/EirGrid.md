**EirGrid** is [[Ireland]]'s state-owned electricity transmission system operator, managing the high-voltage grid and responsible for grid development planning. Works alongside [[ESB]] Networks (distribution) under [[CRU]] regulation.

| Field | Value |

| HQ | Dublin |

| Grid | High-voltage transmission |

Expected to detail implementation rules for the CRU's December 2025 data centre policy (80% renewables requirement) by end of March 2026.
