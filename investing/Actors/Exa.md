#actor #ai #search #agentic #private #usa

Exa (formerly Metaphor Systems) is an AI-native web search company offering a neural/embeddings-based search API for agents and RAG applications, emphasizing semantic relevance for open-ended and research-style queries. It is a direct competitor to [[Parallel Web Systems]] and [[Tavily]] in the agent-search-infrastructure category and is benchmarked head-to-head against Parallel.
