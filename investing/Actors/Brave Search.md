Brave Search is the independent search engine and search API built by Brave Software (the privacy browser company founded by [[Brendan Eich]], creator of JavaScript and co-founder of [[Mozilla]]), notable for running its own index rather than reselling [[Google]] or [[Microsoft]] results. Its API competes with [[Parallel Web Systems]], [[Exa]], [[Tavily]], and [[Firecrawl]] for AI-agent search traffic, and in independent benchmarking led on both answer quality and latency — beating Parallel's deep-research latency by roughly an order of magnitude (~669ms vs ~13.6s).
