# MSCI

Global index provider. NYSE: MSCI.

## Key indices

| Index | Description |
|-------|-------------|
| MSCI World | Developed markets |
| MSCI ACWI | All Country World Index |
| MSCI Emerging Markets | EM benchmark |
| MSCI China | China equities |
| MSCI [[Asia]] Pacific IT | Asian tech |

## Why it matters

Index inclusion/exclusion drives passive flows. MSCI upgrades (e.g., frontier → emerging → developed) trigger mandatory buying from index funds and ETFs tracking these benchmarks.
