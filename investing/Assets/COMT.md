# COMT

**COMT** is the iShares GSCI Commodity Dynamic Roll Strategy ETF, a broad commodity futures ETF from [[BlackRock]] / iShares. It tracks a dynamic-roll commodity strategy rather than the [[Bloomberg Commodity Index]], but it is a useful local comparator for diversified commodity exposure.

## Use Case

COMT belongs in the commodity-proxy set because it gives broad futures exposure with dynamic roll mechanics. It is not a BCOM tracker and should not be used as the Bloomberg-linked line; use [[COMB]] for that. Use COMT when comparing how different commodity ETF wrappers respond to the same futures-market regime.
