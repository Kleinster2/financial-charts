#concept #privacy

**Private Cloud Compute** — [[Apple]]'s cloud AI infrastructure designed to process AI requests with on-device-level privacy guarantees. Extends Apple's privacy model to server-side AI workloads.

---

## Quick stats

| Metric | Value |
|--------|-------|
| Owner | [[Apple]] |
| Purpose | Privacy-preserving cloud AI inference |
| Key feature | No persistent data storage on servers |
| Used by | [[Siri]], [[Apple Intelligence]] |

---

## Why it matters

Allows [[Apple]] to run AI models too large for on-device processing while maintaining privacy commitments. Central to the [[Apple-Google AI deal]] — Tim Cook confirmed the [[Gemini]]-powered [[Siri]] will run through Private Cloud Compute, ensuring [[Google]] does not access user data.
