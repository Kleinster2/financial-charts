#concept #investing #event #strategy

**Catalyst investing** — Focus on events that unlock value. Earnings, M&A, spin-offs, management change. Reduces timing risk.

---

## Catalyst types

| Category | Examples |
|----------|----------|
| **Earnings** | Beats, guidance raises |
| **Corporate actions** | M&A, spin-offs, buybacks |
| **Management** | New CEO, activist involvement |
| **Regulatory** | Approvals, settlements |
| **Macro** | Rate cuts, policy changes |
| **Technical** | Index inclusion, short squeeze |

---

## Catalyst timeline

| Horizon | Examples |
|---------|----------|
| Near-term (<3 mo) | Earnings, FDA decisions |
| Medium-term (3-12 mo) | Spin-offs, M&A close |
| Long-term (1+ yr) | Turnarounds, secular shifts |

---

## Earnings catalysts

| Type | Signal |
|------|--------|
| Beat + raise | Strong momentum |
| Beat + guide down | One-time strength |
| Miss + guide up | Temporary issue |
| Miss + guide down | Thesis damage |

---

## Corporate action catalysts

| Action | Value unlock |
|--------|--------------|
| Spin-off | Sum-of-parts discount |
| Split-off | Forced selling creates opportunity |
| Buybacks | Share count reduction |
| Special dividend | Returning excess cash |
| Going private | Premium paid |

---

## Activist catalysts

| Action | Potential outcome |
|--------|-------------------|
| 13D filing | Board pressure begins |
| Board seats | Strategy influence |
| Strategic review | Sale or restructuring |
| Management change | Turnaround |

---

## Regulatory catalysts

| Event | Impact |
|-------|--------|
| FDA approval | Biotech value unlock |
| Patent expiration | Generic competition |
| Merger approval | Deal closes |
| Litigation settlement | Uncertainty removed |

---

## Evaluating catalysts

| Factor | Question |
|--------|----------|
| Probability | How likely? |
| Magnitude | How much upside? |
| Timing | When? |
| Downside | If it fails? |
| Consensus | Priced in? |

---

## Risk considerations

| Risk | Mitigation |
|------|------------|
| Catalyst fails | Position sizing |
| Delayed timing | Patience, cost of carry |
| Priced in already | Variant perception needed |
| Thesis wrong | Stop loss |
