#concept #trade #uk #brexit

**UK post-Brexit FTAs** — Britain's independent trade policy after leaving [[EU]]. Rollover deals plus new agreements with [[Australia]], NZ. CPTPP accession 2023.

---

## Overview

Post-Brexit trade strategy:

| Phase | Status |
|-------|--------|
| **Rollover deals** | 70+ countries (copy [[EU]] FTAs) |
| **New FTAs** | [[Australia]], New Zealand |
| **CPTPP** | Acceded July 2023 |
| **US FTA** | Stalled (not priority for US) |
| **[[India]]** | Negotiating |

### UK-[[Australia]] FTA (2023)

| Metric | Value |
|--------|-------|
| Signed | Dec 2021 |
| In force | May 2023 |
| First new FTA | Post-Brexit |

| Sector | Terms |
|--------|-------|
| Beef | Quota → full liberalization (15 yr) |
| Lamb | Quota → full liberalization (10 yr) |
| Wine | Duty-free |
| Services | Strong access |
| Mobility | Youth mobility scheme |

**Controversy:** UK farmers opposed beef/lamb opening.

| Metric | Value |
|--------|-------|
