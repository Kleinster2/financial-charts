# Collaborative Combat Aircraft

US military program to develop autonomous drone wingmen that fly alongside manned fighters. The defining shift from expensive, exquisite platforms to affordable, attritable unmanned systems. Target: 1,000-2,000 CCAs teaming with F-35s and the next-gen F-47, at roughly one-third the cost of an F-35 (~$25-30M per airframe vs ~$80M).

---

## Synthesis

CCA is the largest US drone procurement program in history by budget — ~$1.39B in FY2025 alone, with ~$996.5M requested for FY2027 as the first major production year. Three contractors are competing for Increment 1 (air-to-air mission): [[General Atomics]] (YFQ-42A, first flight Aug 2025), [[Anduril]] (YFQ-44A Fury, first flight Oct 2025, production started Mar 2026 at Arsenal-1 in Ohio), and [[Northrop Grumman]] (YFQ-48A Talon Blue, self-funded late entrant designated Dec 2025). The program's speed is unprecedented for major defense acquisition — Anduril went from clean-sheet design to flying prototype in 556 days, and from first flight to production start in five months.

Increment 2 is broader: nine classified vendors received concept refinement contracts (Dec 2025), spanning affordable/attritable to higher-end designs. The USMC is running a parallel program (MUX TACAIR) with [[Northrop Grumman]] as prime and [[Kratos]] providing the XQ-58 Valkyrie airframe ($231.5M OTA, Jan 2026).

Internationally, [[Australia]]'s MQ-28A Ghost Bat ([[Boeing]]) achieved the first CCA live-fire missile test (AIM-120 AMRAAM, Dec 8, 2025) and [[Germany]] is evaluating it for the Luftwaffe. The European [[FCAS]]/SCAF program (France-Germany-Spain) is teetering near cancellation due to workshare disputes.

## Program structure

### Increment 1 — air-to-air combat

| Metric | Value |
|--------|-------|
| Mission | Missile truck / escort for manned fighters (F-35, F-47) |
| Secondary missions | EW, ISR |
| Planned buy | 100-150 aircraft |
| Target unit cost | ~$25-30M (~1/3 of F-35) |
| IOC target | End of decade (~2028-2030) |
| Estimated procurement | ~$4.5B (Shephard); total program ~$8.9B |

### Increment 2 — broader scope

- Nine vendors received concept refinement contracts (Dec 22, 2025); names classified
- Designs span affordable/attritable to higher-end
- Then-Secretary Kendall: should not be "exquisite" but more capable than Increment 1, with 20-30% additional cost acceptable
- Expected missions: ground attack, electronic warfare

## Increment 1 competitors

### [[General Atomics]] — YFQ-42A "Dark Merlin"

| Spec | Value |
|------|-------|
| First flight | August 27, 2025 (California) |
| Derived from | XQ-67A Off-Board Sensing Station |

| Range | 700+ nm (exceeds F-22's 590nm and F-35A's 670nm) |
| Autonomy | Trained across 5+ years on MQ-20 Avenger platform |

### [[Anduril]] — YFQ-44A "Fury"

| Spec | Value |
|------|-------|
| First flight | October 31, 2025 (Victorville, CA) |
| Design timeline | Clean-sheet to flying prototype in 556 days |
| Configuration | Single-engine, jet-powered, semi-autonomous |
| Autonomy | Taxi to landing, preplanned mission profile, human supervision on return |
| Production | Started March 2026 at Arsenal-1 (Pickaway County, Ohio), 3 months ahead of schedule |
| Arsenal-1 | ~$1B facility. Also manufactures Roadrunner and Barracuda. Initial capacity: 50 Fury/year, target 150/year. 250 employees by end-2026, target 4,000 over decade |

| Spec | Value |
|------|-------|

| Design timeline | Designed, built, readied to fly in under 2 years |

## XQ-67A — technology precursor

[[General Atomics]]' XQ-67A Off-Board Sensing Station (OBSS) for AFRL. First flight February 28, 2024 (Gray Butte, Palmdale, CA). Precursor to the YFQ-42A CCA — "common core system." Successfully demonstrated government-owned reference autonomy with tactical datalink (Jul 2025). The R&D platform; YFQ-42A is the production derivative.

## International programs

### [[Australia]] — MQ-28A Ghost Bat ([[Boeing]])

| Metric | Value |
|--------|-------|
| Rename | Loyal Wingman → MQ-28A Ghost Bat (Mar 2022) |
| Test flights | 100+ by Mar 2025 |
| First live-fire | Dec 8, 2025 — AIM-120 AMRAAM destroyed Phoenix target drone, paired with E-7 Wedgetail + F/A-18F |
| Order (Block 2) | 6 aircraft + Block 3 prototype, A$1.4B (~US$930M), Dec 9, 2025 |
| Additional order | 7 more (6 Block 2 + first Block 3), A$754M (~US$501M) |
| Target unit cost | ~10% of crewed fighter (~$8-10M) |
| Length | 38 ft (11.7m) |
| Range | 2,000+ nm |
| Operational | Targeted 2028. Block 3 designed for international customers |

### [[Germany]] — MQ-28 evaluation

Defense Minister Boris Pistorius said MQ-28 Ghost Bat is "under consideration" (Mar 2026). If selected, [[Rheinmetall]] would serve as system manager for integration, maintenance, logistics.

### European FCAS/SCAF (France-Germany-Spain)

Led by [[Dassault Aviation]] (NGF fighter), [[Airbus]] (remote carriers + combat cloud), [[Indra Sistemas]]. Remote Carriers: uncrewed drones of various sizes teaming with manned 6th-gen fighter. Phase 2 repeatedly delayed. As of Feb 2026: "teetering on brink of cancellation" due to France-Germany workshare disputes.

[[Airbus]] separately pitching a stealthy loyal wingman for Luftwaffe Eurofighters — uncrewed demo flights planned 2026, operational target 2029. Also partnering with [[Saab]] on a loyal wingman concept for Germany/Sweden.

### UK — LANCA / Project Mosquito

Lightweight Affordable Novel Combat Aircraft concept (started 2015). Project Mosquito demonstrator (Spirit AeroSystems Belfast + [[Northrop Grumman]]): cancelled June 2022, did not fly. No replacement announced.
