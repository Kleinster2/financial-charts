# x402

#concept #payments #agent-economy

**x402** — Open standard for HTTP-native micropayments. Clients receive HTTP 402 (Payment Required) and pay on-demand per request. Designed for the agent era: AI agents browsing the web can pay for content without subscriptions or ad models.

Spec: [x402.org](https://www.x402.org/)

Built into [[EmDash]] ([[Cloudflare]]'s CMS) as default monetization layer.
