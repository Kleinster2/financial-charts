**Valuation multiples** — Ratios comparing price to fundamentals. P/E, EV/EBITDA, P/S. Relative valuation tool.

---

## Common multiples

| Multiple | Formula | Best for |
|----------|---------|----------|
| **P/E** | Price / EPS | Profitable companies |
| **EV/EBITDA** | EV / EBITDA | Capital-intensive |
| **P/S** | Price / Revenue | Growth, unprofitable |
| **P/B** | Price / Book | Financials, asset-heavy |
| **P/FCF** | Price / FCF | Cash generators |
| **EV/Revenue** | EV / Revenue | Early-stage |

---

## P/E ratio

| P/E Range | Interpretation |
|-----------|----------------|
| <10 | Cheap or distressed |
| 10-15 | Value |
| 15-20 | Fair |
| 20-30 | Growth premium |
| >30 | High growth or expensive |

**Forward P/E** uses estimated earnings.
**Trailing P/E** uses last 12 months.

---

## EV/EBITDA

| Range | Interpretation |
|-------|----------------|
| <6x | Cheap |
| 6-10x | Fair |
| 10-15x | Premium |
| >15x | Expensive |

**Advantage**: Capital structure neutral.

## Growth-adjusted multiples

| Multiple | Formula |
|----------|---------|
| **PEG** | P/E / Growth rate |
| **EV/EBITDA/G** | EV/EBITDA / Growth |

| PEG | Interpretation |
|-----|----------------|
| <1 | Cheap for growth |
| 1-2 | Fair |
| >2 | Expensive |

---

## Enterprise value

**EV = Market Cap + Debt - Cash**

| Component | Impact |
|-----------|--------|
| Market cap | Equity value |
| + Total debt | Creditor claims |
| - Cash | Offsetting asset |
| = EV | Total firm value |

---

## When to use each

| Situation | Multiple |
|-----------|----------|
| Comparing peers | EV/EBITDA |
| Profitable companies | P/E |
| Unprofitable growth | EV/Revenue, P/S |
| Asset-heavy | P/B |
| [[Banks]] | P/B, P/E |
| [[REITs]] | P/FFO |
| Cash machines | P/FCF |

---

## Pitfalls

| Issue | Problem |
|-------|---------|
| Earnings manipulation | P/E distorted |
| One-time items | Adjust to normalized |
| Different accounting | GAAP vs. non-GAAP |
| Growth differences | Can't compare 5% to 30% |
| Cyclical peaks | Earnings inflated |
