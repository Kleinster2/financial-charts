#concept #investing #management #strategy

**Capital allocation** — How companies deploy cash. Key CEO skill. Buybacks, dividends, M&A, reinvestment.

---

## Capital allocation options

| Use | Description |
|-----|-------------|
| **Reinvest** | Organic growth, R&D, capex |
| **M&A** | Acquisitions |
| **Dividends** | Return to shareholders |
| **Buybacks** | Share repurchases |
| **Debt paydown** | Deleverage |
| **Cash hoard** | Optionality |

---

## Decision framework

| Factor | Consider |
|--------|----------|
| ROIC | Returns on reinvestment |
| Growth options | Organic opportunities |
| Valuation | Stock price vs. intrinsic |
| Balance sheet | Leverage capacity |
| Shareholder base | Income vs. growth focus |

---

## Return hierarchy

| Preference | When |
|------------|------|
| Reinvest | ROIC > WACC |
| Buybacks | Stock undervalued |
| M&A | Synergies > premium |
| Dividends | Stable, mature business |
| Cash | Uncertainty, opportunities |

---

## Buyback quality

| Good buybacks | Bad buybacks |
|---------------|--------------|
| Below intrinsic value | At peak valuations |
| Funded from cash flow | Debt-funded |
| Consistent over time | One-time around events |
| Reduce share count | Offset dilution only |

---

## M&A track record

| [[Signal]] | Quality |
|--------|---------|
| Small, bolt-on | Usually value-creating |
| Large, transformational | Usually value-destroying |
| Cash deals | Disciplined |
