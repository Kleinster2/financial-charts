# MCP

Model Context Protocol — [[Anthropic]]'s open standard for connecting LLMs to external data sources and tools. Bidirectional, local-first protocol that any model can implement.

---

## Quick stats

| Detail | Value |
|--------|-------|
| Originator | [[Anthropic]] |
| Type | Open specification |
| Launch | Late 2024 |
| Adoption | Broad — [[Claude]], third-party models, IDE integrations |

MCP is the connectivity layer in Anthropic's platform stack: MCP (tools) → [[Claude Skills]] (knowledge) → Code Execution (environment).

See [[AI extensibility]] for broader context on extensibility approaches.
