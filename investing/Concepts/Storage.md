#concept #infrastructure

**Storage** — Enterprise data storage systems. [[EMC]] dominated for decades. Now consolidated under [[Dell|Dell Technologies]], [[HPE]], [[NetApp]].

---

## Why storage matters

Data growth drives infrastructure spend:

- **Enterprise data**: Doubling every 2-3 years
- **AI/ML**: Training data requires massive storage
- **Compliance**: Retention requirements expanding
- **Hybrid cloud**: On-prem + cloud storage integration

## Storage types

| Type | Use case | Players |
|------|----------|---------|

| **File** | Shared files, NAS | NetApp, Dell EMC Isilon |
| **Object** | Unstructured data, cloud | AWS S3, Dell ECS |
| **Flash/SSD** | High performance | Pure Storage, Dell, NetApp |
