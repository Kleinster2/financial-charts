# Urban digitization

#concept #infrastructure #smartcities #iot

**Urban digitization** — Retrofitting existing cities with digital infrastructure (sensors, connectivity, AI) vs building greenfield smart cities. Different challenges, proven ROI, and dominant model globally. Sister concept to [[Smart cities]].

---

## Retrofit vs greenfield

| Aspect | Retrofit (established cities) | Greenfield (new builds) |
|--------|-------------------------------|-------------------------|
| **Examples** | London, NYC, Singapore, Tokyo | [[NEOM]], Songdo, Xiong'an |
| **Approach** | Layer onto legacy | Design from scratch |
| **Timeline** | Incremental (decades) | Big bang (risky) |
| **Cost model** | Spread, ROI-driven | Massive upfront |
| **Risk** | Fragmentation | Megaproject failure |
| **Track record** | Proven | Mixed ([[NEOM]] struggling) |

**Key insight:** Most smart city success stories are retrofits, not greenfield. [[NEOM]]-style megaprojects have high failure rates.

---

## Why retrofit works

| Factor | Explanation |
|--------|-------------|
| **Existing demand** | Real users, real problems |
| **Iterative** | Test, learn, scale |
| **ROI visible** | Barcelona saved €92M |
| **Political feasible** | Incremental budgets |
| **Infrastructure base** | Fiber, roads, transit exist |

## Common retrofit patterns

### 1. [[Transport]] digitization

| Component | Examples |
|-----------|----------|
| Contactless payments | Oyster (London), OMNY (NYC) |
| Congestion pricing | London (£18/day), Singapore |
| Real-time tracking | All major metros |
| Demand management | Dynamic pricing, routing |

### 2. Utility optimization

| Component | Impact |
|-----------|--------|
| Smart meters | 19,500 in Barcelona |
| Water sensors | 25% conservation (Barcelona) |
| Smart lighting | LED + sensors |
| Grid management | Demand response |

### 3. Citizen services

| Component | Examples |
|-----------|----------|
| 311/service apps | NYC 311, Singapore OneService |
| Digital ID | SingPass (Singapore) |
| Cashless payments | PayNow (Singapore) |
| Public Wi-Fi | LinkNYC, Barcelona 1,000+ hotspots |

### 4. Safety & resilience

| Component | Examples |
|-----------|----------|
| CCTV networks | London (extensive), Tokyo |
| Early warning | Tokyo earthquake (80 sec advance) |
| Flood monitoring | Tokyo storm surge AI |
| Emergency response | Integrated dispatch |

## Challenges

| Challenge | Detail |
|-----------|--------|
| **Legacy integration** | Old systems don't talk to new |
| **Vendor lock-in** | Proprietary platforms |
| **Privacy concerns** | CCTV, data collection |
| **[[Cybersecurity]]** | Expanded attack surface |
| **Digital divide** | Not all residents connected |
