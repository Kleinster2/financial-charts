**AUKUS** — Australia-UK-US security pact. Nuclear submarines for Australia. Indo-Pacific focus. Counters China.

---

## What is AUKUS

| Element | Description |
|---------|-------------|
| Members | Australia, UK, US |
| Announced | September 2021 |
| Focus | Indo-Pacific security |
| Centerpiece | Nuclear-powered submarines for Australia |

---

## Pillar 1: Submarines

| Phase | Timeline | Description |
|-------|----------|-------------|
| Phase 1 | Now-2027 | US/UK submarine visits to Australia |
| Phase 2 | 2030s | Australia buys 3-5 Virginia-class (US) |
| Phase 3 | 2040s | SSN-AUKUS jointly developed |

Australia will operate nuclear-powered (not nuclear-armed) submarines.
