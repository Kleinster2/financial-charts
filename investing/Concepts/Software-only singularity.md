# Software-only singularity

A software-only singularity is a proposed AI takeoff path in which AI systems automate AI R&D, improve the software of future AI systems, and thereby create a recursive loop where algorithmic progress gets faster and faster without requiring a matching hardware discontinuity.

## Core parameter

The Forethought framework uses `r` as the central shorthand:

| Condition | Meaning |
|-----------|---------|
| `r > 1` | Each doubling of software capability requires less than a doubling of cumulative R&D effort; progress accelerates |
| `r = 1` | Exponential growth; each software doubling requires a proportional doubling of effort |
| `r < 1` | "Fizzle" regime; progress can still be fast, but each additional doubling takes more effort |

[[Ryan Greenblatt]]'s May 2026 contribution is to separate this threshold from the investment-relevant question. A true software-only singularity requires sustained `r > 1`, but [[AI R&D automation]] could still produce a large one-time speed-up and higher returns to compute in an `r < 1` regime.
