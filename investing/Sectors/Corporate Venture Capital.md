## Strategic vs financial

| Type | Goal | Trade-offs |
|------|------|------------|
| **Strategic** | Ecosystem building | May scare other investors |
| **Financial** | Returns | Less strategic value |
| **Hybrid** | Both | Most common today |

---

## CVC advantages

| Advantage | Description |
|-----------|-------------|
| Distribution | Access to parent's customers |
| Technical | Engineering resources |
| Credibility | Validation from big name |
| M&A path | Potential acquirer |

---

## CVC disadvantages

| Disadvantage | Description |
|--------------|-------------|
| Competitor concerns | Scares away rivals |
| Slow decisions | Corporate bureaucracy |
| Strategy shifts | Parent priorities change |
| Information rights | Competitive intel concerns |
