# REITs

## REIT structure

| Feature | Detail |
|---------|--------|
| Tax status | Pass-through (no corporate tax) |
| Dividend requirement | 90%+ of taxable income |
| Yield | Typically 3-6% |
| Interest sensitivity | Inverse correlation |

---

## Key metrics

| Metric | What it measures |
|--------|------------------|
| **FFO** | Funds from operations (cash flow) |
| **AFFO** | Adjusted FFO (maintenance capex) |
| **NAV** | Net asset value |
| **Cap rate** | NOI / property value |
| **Occupancy** | % leased |
| **Same-store NOI** | Organic growth |

---

## Interest rate sensitivity

| Environment | REIT impact |
|-------------|-------------|
| Rising rates | Headwind (higher cap rates) |
| Falling rates | Tailwind (valuation expansion) |
| Inflation | Mixed (rent escalators vs costs) |
