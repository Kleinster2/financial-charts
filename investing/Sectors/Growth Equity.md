## Investment characteristics

| Trait | Description |
|-------|-------------|
| Stage | Series C through pre-IPO |
| Check size | $50M - $1B+ |
| Ownership | 5-20% (minority) |
| Board seats | Often no (esp. DST, Tiger) |
| Hold period | 3-7 years |
| Exit | IPO or secondary sale |

---

## Growth vs VC vs PE

| Aspect | VC | Growth Equity | PE |
|--------|-----|---------------|-----|
| Stage | Early | Late | Mature |
| Control | Minority | Minority | Control |
| Profitability | Pre-revenue OK | Path to profit | Profitable |
| Leverage | None | None | Yes |
| Check size | $1-50M | $50M-1B | $500M+ |
