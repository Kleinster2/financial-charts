## Activist playbook

| Step | Action |
|------|--------|
| 1 | Quietly build stake (5%+ triggers 13D) |
| 2 | Private engagement with management |
| 3 | Public letter if no progress |
| 4 | Seek board seats |
| 5 | Proxy fight if necessary |
| 6 | Litigation (Elliott specialty) |

## Activist defenses

| Defense | Description |
|---------|-------------|
| Poison pill | Dilute activist stake |
| Staggered board | Can't replace all at once |
| Supermajority | High bar for changes |
| Golden parachutes | Expensive to fire mgmt |
| White knight | Find friendly buyer |
