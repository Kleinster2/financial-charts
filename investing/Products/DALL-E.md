# DALL-E

[[OpenAI]]'s image generation family. Integrated into [[ChatGPT]]. DALL-E 3 (2023) set standard for prompt understanding. Being superseded by GPT Image models.

## Quick stats

| Metric | Value |
|--------|-------|
| Current version | GPT Image 1.5 (2025) |
| Previous | DALL-E 3 (Oct 2023) |
| Access | ChatGPT, API |
| Resolution | Up to 1792px |
| Key strength | Text rendering, prompt accuracy |

---

## Version history

| Model | Release | Key changes |
|-------|---------|-------------|
| DALL-E | Jan 2021 | Original text-to-image |
| **DALL-E 2** | Apr 2022 | Higher resolution, inpainting |
| **DALL-E 3** | Oct 2023 | ChatGPT integration, superior prompt understanding |
| GPT Image 1 | 2025 | Multimodal architecture |
| GPT Image 1 Mini | 2025 | Cost-optimized |
| **GPT Image 1.5** | Late 2025 | Current flagship, GPT-5 architecture |
