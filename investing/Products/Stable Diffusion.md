# Stable Diffusion

[[Stability AI]]'s open-weights image generation family. Most widely deployed AI image model. SD 3.5 (2024) current, SD4 (2026) announced. Free for <$1M revenue.

## Quick stats

| Metric | Value |
|--------|-------|
| Current version | SD 3.5, SD4 (2026) |
| License | Community (free <$1M), Enterprise |
| Key strength | Open weights, self-hosting |
| Resolution | Up to 1 megapixel |
| Access | Local, API, cloud partners |
