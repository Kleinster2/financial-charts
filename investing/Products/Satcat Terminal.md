#product #space #saas #intelligence

**Satcat Terminal** — [[Kayhan Space]]'s investor/insurer-facing intelligence platform. Launched March 20, 2026. AI-powered reference for 36,000+ tracked orbital objects, including 11,000+ satellites with daily-refreshed trajectories. The product framing — explicit Bloomberg-terminal analog for orbital data — signals that orbital activity has reached the scale where investors, insurers, and enterprises need a standardized intelligence layer to underwrite risk and opportunity.

## Features

| Feature | Description |
|---|---|
| AI chat agent | Plain-language queries against the orbital catalog |
| Constellation health monitoring | Live trajectory and operational status |
| Expansion cadence | Track satellite production rates and orbit insertions |
| Anomalous behavior detection | Flag satellite events (deorbits, position changes, signal loss) |
| Daily trajectory refresh | 11,000+ satellites updated daily |
| Coverage breadth | Both publicly available and partner-sourced orbital data |

The AI chat layer is the product's differentiator — the underlying catalog is partially publicly available, but turning it into an analyst-grade research tool that doesn't require orbital mechanics expertise is the moat.

## Why Satcat Terminal matters

Per [[Space Capital]] (Chad Anderson, Apr 26, 2026): "To manage all this growth in orbit, Kayhan launched the Satcat Terminal this quarter. A Bloomberg terminal analog for orbital activity gives investors and insurers a new tool to query constellation health, expansion cadence, and anomalous satellite behavior. The orbital economy now has a data terminal. What comes next is pricing, risk models, and derivatives."

The structural read: as orbital activity scales (Starlink at 10K simultaneous satellites, [[Project Sunrise]] filing 51,600, SpaceX FCC filing 1M), the *information* layer becomes the choke point. Investors pricing equity, insurers underwriting risk, and operators avoiding collisions all need a shared data backbone — and one provider tends to win the standardized layer. Bloomberg's analog history matters: terminal subscriptions became a $10B/year business because the data moat compounds with adoption.
