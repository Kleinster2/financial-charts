# OpenAI GPT-4.5

#ai #openai #products

Summary
- Release: 2025-02-27 (research preview)
- Access: ChatGPT Pro at launch; later Plus/Team; API access until 2025-07-14; removed from standard ChatGPT/most API tiers after GPT-5 (Aug 2025), retained for Pro as legacy
- Positioning: Incremental over GPT‑4o; not a “reasoning” model (no default CoT); emphasized more natural conversation, lower hallucinations
- Pricing (API, at launch): ~$75/M input, ~$150/M output tokens (vs GPT‑4o $2.5/$10)

Capabilities (reported)
- Lower hallucinations vs GPT‑4o
- Stronger conversational nuance and alignment to user intent
- Training: unsupervised pretraining + supervised fine‑tuning + RLHF on Azure
- In ChatGPT: search + Canvas/file/image inputs at launch (no voice/video/screenshare initially)

Benchmarks/coverage
- Outperformed GPT‑4o on MMLU across 15 languages (per system card)
- Review consensus: meaningful but not a giant leap; “~20% better on specific tasks” (Axios quoting Aaron Levie)

Lifecycle/Deprecation
- 2025‑04‑14: API phase‑out announcement (effective 2025‑07‑14) in favor of GPT‑4.1
- 2025‑08‑07: With GPT‑5 release, GPT‑4.5 removed for most users; remains for Pro under “Legacy Models” (per reporting)
