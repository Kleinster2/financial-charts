# Dream Machine

[[Luma AI]]'s video generation platform. Ray3 (Dec 2025) enables video-to-video with performance preservation. First HDR video model.

## Quick stats

| Metric | Value |
|--------|-------|
| Current model | Ray3 (Dec 2025) |
| Max duration | 60 seconds |
| Key strength | Motion coherence, HDR |
| Access | Web, API |
