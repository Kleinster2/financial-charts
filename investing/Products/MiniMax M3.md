# MiniMax M3

MiniMax M3 is [[MiniMax]]'s June 2026 frontier model release, positioned around coding, agentic work, 1M-token context, and native multimodal input.

## The story

MiniMax is using M3 to move the China AI-lab narrative from "cheap model access" to "frontier capability plus long-context infrastructure." The model release matters for the investing vault because it landed almost simultaneously with MiniMax's proposed RMB-share / [[STAR Market]] route: model capability, API economics, and access to public capital are now part of the same story.

## Architecture and release facts

MiniMax says M3 uses MiniMax Sparse Attention to make 1M-token context usable for long-range coding, agent tasks, and video understanding. Its model page describes M3 as supporting up to 1M tokens with a guaranteed 512K minimum, while the launch post describes native multimodality and desktop-operation capabilities.

The model is framed as open-weight by MiniMax, but the investing note should verify the exact weights, license, and release mechanics once the downloadable weights and technical report are actually available.

## Benchmark posture

MiniMax claims M3 reaches frontier-level results on coding, agentic, SVG, document, and multimodal benchmarks. For now, the correct treatment is "MiniMax claims" rather than benchmark fact. The market signal is that MiniMax is trying to sell the same package Western labs are selling - coding, agents, long context, multimodality - with China pricing and public-market accessibility.

## Investing read-through

M3 strengthens MiniMax's public-equity narrative in three ways:

- It gives 0100.HK a current product catalyst beyond the January IPO and February M2.5 token-volume story.
- It reinforces [[China AI Tigers]] as a live public-market cohort rather than a purely private VC category.
- It turns long-context efficiency into a capital-market question: if sparse attention materially lowers inference cost at large context windows, MiniMax's monetization can be judged on token economics, not just benchmark rankings.
