# Sora

[[OpenAI]]'s video generation model. "The GPT-1 moment for video." Sora 2 (2025) generates up to 25 seconds with synchronized audio. Launched dedicated iOS app.

## Quick stats

| Metric | Value |
|--------|-------|
| Current version | Sora 2 |
| Max duration | 20-25 seconds |
| Resolution | Up to 1080p |
| API launch | Sep 30, 2025 |
| ARR | Part of ChatGPT Pro ($200/mo) |
| Access | Plus ($20) and Pro ($200) only |

| Model | Release | Key changes |
