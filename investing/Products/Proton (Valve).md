# Proton (Valve)

#product #gaming #software #compatibility

Proton is [[Valve]]'s compatibility layer that lets Windows-first PC games run on [[SteamOS]] and other [[Linux]] environments.

| Strategic purpose | Makes the Steam library portable across non-Windows Steam hardware |

---

## Why it matters

Proton is the hidden asset behind Valve's hardware strategy. The first Steam Machines failed partly because Linux gaming compatibility was not ready. Proton changes the adoption curve: developers do not need to port every game before a [[SteamOS]] device has a useful library. The remaining friction is concentrated in anti-cheat, secure boot, driver behavior, and edge-case performance rather than basic game availability.
