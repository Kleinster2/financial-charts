# Llama

[[Meta]]'s open-weights LLM family. Most influential open-source AI model. Llama 4 (Apr 2025) has 10M token context (Scout). Free commercial use under Meta license.

## Quick stats

| Metric | Value |
|--------|-------|
| Current version | Llama 4 (Apr 2025) |
| Largest model | Behemoth (2T params, upcoming) |
| Context | Up to 10M tokens (Scout) |
| Languages | 8+ |
| License | Meta Llama License (permissive) |
| Key strength | Ecosystem, fine-tuning |
