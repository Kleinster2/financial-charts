# Google Antigravity

Google Antigravity — [[Google]]'s agent-first development and execution platform for long-horizon AI tasks. Google I/O 2026 positioned Antigravity 2.0 as both a developer product and the harness behind [[Gemini Spark]]: the same control layer that lets coding agents plan, inspect, and act can also run personal-agent workflows.

## Why it matters

Antigravity is the bridge between Google's developer tooling and its consumer-agent strategy. If it works, Google gets one harness for code agents, app generation, [[Google Workspace]] automation, and personal-agent execution. That creates a common control surface for agent reliability, tool permissions, and long-running task state.
