Bitbucket is [[Atlassian]]'s source code repository hosting service and CI/CD platform, supporting Git-based version control for software development teams. Originally launched in 2008 as a [[Mercurial]]-only host, it was acquired by [[Atlassian]] in 2010 and pivoted to Git. The product integrates tightly with [[Atlassian]]'s [[Jira]] and [[Confluence]] tools, forming the developer workflow backbone of the Atlassian ecosystem.

Bitbucket competes primarily with [[GitHub]] ([[Microsoft]], acquired 2018 for $7.5B) and [[GitLab]] (GTLB). It is a distant third in market share — GitHub dominates with 100M+ developers, while Bitbucket's installed base skews toward teams already committed to the Atlassian stack. Atlassian has de-emphasized Bitbucket Server (self-hosted), ending its sale in 2024 as part of the broader cloud migration strategy. Bitbucket Cloud and Bitbucket Data Center remain active products. The platform's CI/CD offering (Bitbucket Pipelines) competes with [[GitHub Actions]] and [[GitLab CI]].
