# Yi

[[01.AI]]'s LLM family. Founded by Kai-Fu Lee. Yi-Lightning (Oct 2024) matches GPT-4o at 14¢/M tokens. Pivoted to enterprise solutions using [[DeepSeek]] models (Mar 2025).

## Quick stats

| Metric | Value |
|--------|-------|
| Current version | Yi-Lightning (Oct 2024) |
| Training | 3T tokens multilingual |
| Key strength | Cost efficiency |
| Status | Pivoted to enterprise (Mar 2025) |
