# Steam Link

#product #gaming #streaming #hardware

Steam Link is [[Valve]]'s earlier living-room streaming product, originally a small hardware box and later an app, that streamed games from a PC running [[Steam platform|Steam]] to another screen.

---

## Quick stats

| Metric | Value |
|--------|-------|
| Parent | [[Valve]] |
| Original hardware role | Low-cost PC-to-TV game streaming box |
| Current role | Steam Link app and streaming capability across devices |
| Strategic relevance | Showed a cheaper living-room path that undercut the original 2015 Steam Machine value proposition |

---

## Why it matters

Steam Link helps explain why the original Steam Machines struggled. If a user already had a gaming PC, a low-cost streaming box solved much of the couch problem without buying a new [[SteamOS]] PC. The 2026 [[Steam Machine]] reverses that role: it is powerful enough to be the host PC streaming to [[Steam Deck]], [[Steam Frame]], or any other Steam Link endpoint.
