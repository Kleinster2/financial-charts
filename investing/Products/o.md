# o

[[OpenAI]]'s reasoning model family. Chain-of-thought architecture. Thinks before responding. o1 launched Sep 2024, o3 current flagship.

## Quick stats (o3)

| Metric | Value |
|--------|-------|
| Type | Reasoning model |
| Architecture | Chain-of-thought |
| Key feature | Extended thinking time |
| Release | 2025 |
| Access | ChatGPT Pro, API |

---

## How o-series works

| Aspect | Details |
|--------|---------|
| Thinking | Model reasons step-by-step before responding |
| Tokens | Uses "thinking tokens" not shown to user |
| Time | Takes longer but more accurate on hard problems |
| Cost | More expensive per query |
