# OpenAI GPT-5.4

#ai #openai #products

Summary
- Version: Minor release in the GPT‑5 family ("5.4")
- Release: 2026‑03‑05
- Variants: GPT‑5.4 (core), GPT‑5.4 Thinking (reasoning‑forward), GPT‑5.4 Pro (higher performance)
- Access: Rolling out across ChatGPT (Plus/Team/Pro/Enterprise/Edu per variant), API, and Codex
- Positioning: Focus on agentic use — native computer use (issuing keyboard/mouse actions from screenshots), improved tool/API calling, stronger multi‑step retrieval/synthesis

Notable capabilities (reported)
- Native computer use: write code to operate the computer; issue keyboard/mouse commands in response to screenshots
- Better web use + multi‑round search to identify sources and synthesize answers
- Tool use: higher accuracy/efficiency calling tools and APIs
- Reasoning: GPT‑5.4 Thinking exposes outline of its work for complex queries (web + Android at launch; iOS coming)
- Factuality: OpenAI claims 33% fewer false individual claims vs GPT‑5.2

Context
- Part of broader industry shift toward agentic models and automation of professional workflows (spreadsheets, docs, presentations, coding)
- Sits alongside “Thinking” and “Pro” variants targeting depth of reasoning and maximum performance, respectively
