# Claude Cowork

[[Anthropic]]'s enterprise collaboration agent (Jan 2026). "[[Claude Code]] for the rest of your work." Agentic file management, document creation, multi-step tasks. Part of the [[Claude]] product family.

Native macOS desktop app announced Feb 17 2026 (Windows coming) — reads/interacts with local files, controls mouse/keyboard/browser for multi-step tasks. Extends Cowork from cloud to local desktop.
