# A18 Pro

[[Apple]]'s 2024 high-end mobile system-on-chip, built for the iPhone 16 Pro and 16 Pro Max on [[TSMC]]'s second-generation 3nm process (N3E). A 6-core CPU (2 performance + 4 efficiency), 6-core GPU, and 16-core Neural Engine made it the first Apple silicon to run the launch wave of [[Apple Intelligence]] on device. Superseded by the [[A19]] in the iPhone 17 family.

Not to be confused with the [[A18-M]] — an unrelated autonomous underwater vehicle by [[Exail Technologies]] that shares only the alphanumeric. See [[Apple Silicon]] for Apple's broader chip strategy.
