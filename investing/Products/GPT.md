# GPT

[[OpenAI]]'s general-purpose model family. Powers [[ChatGPT]]. GPT-4 (2023) established frontier model category. GPT-5 current flagship.

| Metric | Value |
|--------|-------|
| Type | Multimodal generalist |
| Context | 128K tokens |
| Modalities | Text, vision, voice |
| Release | 2025 |
| Access | ChatGPT, API |

| Model | Release | Key changes |
