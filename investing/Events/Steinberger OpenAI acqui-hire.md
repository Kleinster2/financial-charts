## The bidding war

Both [[Mark Zuckerberg]] and [[Sam Altman]] personally courted Steinberger. [[Satya Nadella]] met him at an event in Vienna.

### Zuckerberg / Meta

- Reached out via WhatsApp
- Steinberger insisted on calling immediately; Zuckerberg asked for 10 minutes to finish coding
- They argued for 10 minutes about Claude Code vs Codex
- Zuckerberg was actively tinkering with OpenClaw (not just doing corporate diligence)
- Zuckerberg called Steinberger "eccentric, but brilliant"

### Altman / OpenAI

- Steinberger called him "very thoughtful, brilliant"
- Pitch included compute power tied to [[Cerebras]] partnership
- Alignment on mission: agent "even my mum can use" required frontier model access + distribution to billions
- OpenAI had the consumer distribution Meta lacked

### Why OpenAI won

Steinberger's own words: "What I want is to change the world, not build a large company, and teaming up with OpenAI is the fastest way to bring this to everyone."

Non-negotiable condition: OpenClaw stays open source. Chromium model — open core with commercial layers on top.

## Anthropic's strategic error

The core irony: Steinberger was Anthropic's biggest unpaid evangelist. Every one of 180K+ GitHub stars was an implicit endorsement of Claude.

| Anthropic's response | OpenAI's response |
|---------------------|-------------------|
| Trademark C&D letter | "Come build the future with us" |
| Forced 3 name changes in a week | Altman called him "a genius" on X |
| Created chaos enabling account hijacking | Contributed tokens to the project |
| Pushed project toward model-agnosticism | Offered foundation + continued support |
| No public outreach or partnership offer | Coordinated on "OpenClaw" naming |

The forced rename removed Claude's branding from the most popular AI agent on Earth. A project that shipped with `anthropic/claude-opus-4-6` as default became explicitly model-agnostic.

Steinberger was the closest anyone has come to the predicted "one-person billion-dollar company" (a concept Altman, Karpathy, and Amodei have all discussed):

| Metric | Value |

## Competitive context (Feb 2026)

*(Source: Lex Fridman interview, Feb 2026)*

| Product | Status |
|---------|--------|
| [[Anthropic]] Claude Code | **$1B ARR** in 6 months — default coding tool for a generation of developers |
| [[OpenAI]] Codex | Launched as macOS app early Feb 2026 — positioned as "coding surface" combining reasoning models + dev tools |
| Steinberger's role | Self-described "biggest unpaid promoter for Codex" — built OpenClaw using OpenAI models, recorded 3-hour [[Lex Fridman]] episode comparing Codex vs Claude side-by-side |

OpenAI needed Steinberger because Codex was playing catch-up against Claude Code in a market where developer loyalty was becoming sticky. Steinberger's public advocacy for Codex was worth more than any marketing campaign.

### Multi-agent architecture signal

Altman's phrasing — "smart agents interacting with each other to do useful things" — signals interest in multi-agent systems. Aligns with OpenAI's harness engineering case study: 3 engineers used Codex to produce 1,500+ PRs across a million-line codebase with zero human-written code. Extension from coding to personal productivity is the natural next step.

## Significance

### 1. Distribution beats capability in agent wars

Steinberger preferred Claude's reasoning but chose OpenAI for distribution. If the most popular agent framework built on your model picks your competitor because they ship to more consumers, technical superiority doesn't matter.

### 2. Ecosystem champions are irreplaceable

Anthropic sent a C&D when they should have sent a partnership offer. One of the costliest legal-over-strategic decisions in recent AI history.

### 3. Open-source talent as acqui-hire pipeline

New pattern: viral open-source agent builder → frontier lab hire. OpenAI acquiring agent expertise from grassroots rather than building purely in-house.
